# User Management Portal

## ***Technologies used:***

* **Java, SpringBoot**
* **Angular**
* **JavaScript, TypeScript, HTML, CSS**
* **MySQL**
---------------------------------------------------------------------

* **JSON Web Token (JWT)**
* **Authentication / Authorization**
* **Stateless Session Policy**
* **User Security Roles**
* **User Security Permissions**
* **Migration of Brute Force Attack**
* **Password Encryption**
* **Role-based Access Control**
